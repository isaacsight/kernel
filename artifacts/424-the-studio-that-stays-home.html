THE ARGUMENT
論点

THE API IS NOT THE PRODUCT.

The product is the operating discipline around it: price visibility, bounded authority, recoverable jobs, and files a client can actually keep.

Any model can hold the handle.

GALLEY’s engine is plain HTTP on localhost. A terminal agent can call it with curl. An MCP client can hand finished clips to Palmier. A chat model without tools can still draft the request for a human to run. The model is replaceable because the interface is boring on purpose.

That is the useful meaning of model-agnostic: not that every model behaves the same, but that no one vendor owns the studio door.

The gate is part of the craft.

A paid generation is not an invisible implementation detail. It is a purchase. The engine can estimate for free, display the total, enforce a daily cap, persist the job, and return a local receipt. Approval and refusal deserve equal visual weight because both are valid editorial decisions.

The first proof film—thirteen clips, thirteen keyframes, narration, and a Palmier cut—cost about $19.50. That number is more persuasive than an unlimited demo because it lets a client reason about production.

## First-Time Contributors

We especially welcome first-time open-source contributors. Look for issues labeled:
- `good first issue` — small, well-scoped tasks
- `help wanted` — tasks where we'd love community input
- `documentation` — docs improvements (no code required)
- `research` — academic and scientific feature requests

Run `kbot oss find --mentored` to discover mentored issues across all of open source.

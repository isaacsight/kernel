**Body:**
Hey HN,

I built kbot — an open-source (MIT) terminal AI agent that does things differently:

- **787+ built-in tools** — files, git, bash, web search, security scanning, music production, science lab, game dev, and more
- **35 specialist agents** — auto-routes your request to the right expert (coder, researcher, guardian, quant, etc.)
- **Dreams** — after each session, kbot consolidates what it learned about you using local Ollama models. Free. Your preferences and patterns remembered across sessions.
- **Builds its own tools** — if kbot can't do something, it creates the tool at runtime (forge_tool). Sandboxed, persisted, shareable.
- **20 AI providers** — Claude, GPT, Gemini, Grok, DeepSeek, Groq, Ollama, embedded llama.cpp. Switch with one command. BYOK.
- **$0 local mode** — runs fully offline with embedded models. No API key, no data leaves your machine.

The key differentiator: kbot learns. Bayesian skill ratings per agent, pattern extraction from every session, and opt-in collective intelligence so the 1,000th user starts smarter than the 1st.

Install: `npm install -g @kernel.chat/kbot`

# Project Brief

## Project Overview
"Does This Feel Right?" is a modern, static blog built with Python. It features AI-powered content generation, a terminal-based admin interface (TUI), and automated deployment. The project aims to combine the speed and security of a static site with the power of modern AI tools for content creation and management.

## Core Objectives
- **Static Site Generation**: Generate a fast, secure, and SEO-optimized static site from markdown content.
- **AI Integration**: Leverage Gemini, OpenAI, and Anthropic APIs to assist in content creation.
- **Efficient Administration**: Provide a pleasant and efficient TUI for managing content and site settings.
- **Automated Workflows**: Automate testing, security scanning, and deployment via GitHub Actions.
- **Visual Excellence**: Maintain a high standard of design, both for the public-facing blog ("Swiss Console" aesthetic) and the internal tools.

## Key Features
- Custom Python build system.
- Textual-based Admin Dashboard.
- Integrated AI content generation.
- Automated CI/CD pipelines.
- Comprehensive testing and security scanning.

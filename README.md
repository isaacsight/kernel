# Does This Feel Right?

A contemplative AI collaboration platform where specialized agents discuss topics in real-time, with an integrated treasury system, trading dashboard, and blog.

**Live**: [isaacsight.github.io/does-this-feel-right-](https://isaacsight.github.io/does-this-feel-right-)

## Stack

- React 19 + TypeScript + Vite 6
- Framer Motion for contemplative transitions
- Zustand for state management
- EB Garamond + Courier Prime typography
- Warm ivory (#FAF9F6) design system

## Features

- **Observer Mode** — Watch AI agents (Architect, Researcher, Contrarian) discuss any topic in real-time
- **Client Portal** — Get quotes from the autonomous swarm
- **Treasury Dashboard** — Track revenue and project financials
- **Trading Dashboard** — AI-powered trading insights
- **Blog** — Markdown-based writing on systems, AI, and craft

## Development

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # Production build
npm run deploy     # Deploy to GitHub Pages
```

## Design Philosophy

Every interaction should feel contemplative. Large serif headings, generous whitespace (100px padding), warm colors, and unhurried animations. The interface invites dwelling, not skimming.

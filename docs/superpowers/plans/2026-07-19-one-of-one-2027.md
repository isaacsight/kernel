# "One of One" (Spring 2027) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Produce a ~45s vertical (1080x1920 @ 30fps) speculative fashion short per the approved spec at `docs/superpowers/specs/2026-07-19-one-of-one-2027-design.md`, generated via the local fal.ai engine and ElevenLabs audio, assembled and mastered in Palmier Pro.

**Architecture:** All paid generation routes through the existing local video engine (`tools/local-video-server.mjs`, port 5412), which holds FAL_KEY server-side and enforces quotes + spend tracking. One small engine extension adds ElevenLabs sound-effects/music endpoints to the registry. Palmier Pro is the edit bay only. A human greenlight gate sits between prompt authoring and the first paid call.

**Tech Stack:** Node 22 local engine (fal.ai queue API), fal-hosted ElevenLabs audio endpoints, Palmier Pro MCP, ffmpeg two-pass loudnorm, Vitest.

## Global Constraints

- **HARD GATE:** No paid generation (fal.ai or ElevenLabs, video, image, or audio) fires until Isaac approves the greenlight packet (Task 3). Tasks 1–3 must be zero-spend.
- Budget ceiling: **$50 total**; target $30–45. Log every paid call's cost in the greenlight packet's ledger table as you go.
- Deliverable: 1080x1920, 30fps, H.264 + AAC, ~45s.
- Master target: **-16 LUFS integrated / -1.5 dBTP**, two-pass.
- On-screen text only: phone notification (shot 1) and dateline card "SPRING 2027" (shot 10). No captions, no VO.
- Hard cuts only. No transitions, no kinetic typography (house rules, commits ef6660b23 / 23637878a).
- No emojis in any user-visible copy.
- All engine requests go to `http://localhost:5412`; never call fal.ai or ElevenLabs directly with raw keys from this session (`.env` is off-limits per security rules).

export const SFX_PROVIDERS = {
  'elevenlabs-sfx': {
    endpoint: 'fal-ai/elevenlabs/sound-effects/v2',
    usdPerGeneration: 0.10, // VERIFY against fal catalog in Task 1; update if different

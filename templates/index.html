The Simulation

Studio OS is an AI-native operating system for running a solo founder’s entire business as a deterministic simulation.

Architecture

Browser Agents → Orchestrator → Simulation Core → Integrations
  • Runs: Discrete, reproducible execution units of business logic.
  • Worlds: Isolated state containers for simulation contexts.
  • Agents: specialized AI workers with distinct permissions and tools.
  • Resources: External APIs and data sources (GitHub, Stripe, Email).

Components

Frontend: SSG + Vanilla JS. Visualizes repository state.

Backend: Python CORE, Make orchestration. Flat-file JSON/MD.

AI Layer: Gemini 1.5 Pro / Flash. Structured Output schemas.

Infrastructure: Local-first execution. GitHub Pages hosting.

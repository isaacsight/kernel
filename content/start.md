# How This Works

**Does This Feel Right?** is a live operating system for a studio-of-one. It's not a blog, not a portfolio—it's a working system that builds agentic products and patterns in public.

This page explains how to navigate it.

---

## What this studio is

The Studio is a system that turns research into shipped artifacts through agents, patterns, and decision logs.

Unlike a consulting firm or agency, this studio:
- **Runs on its own OS first** — every pattern was battle-tested internally before being published
- **Publishes the infrastructure** — see the actual decision logs, agent configs, and system designs
- **Invites copying** — patterns are built to be forked, not just admired

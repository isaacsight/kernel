# Syntax is Artifact

Writing C++ syntax is like writing Assembly. It is an abstraction layer that is becoming too low-level.
We are moving to a new abstraction layer: **Intent**.

## The End of the For Loop
I haven't written a `for` loop in 6 months.
I write: "Iterate through the users and filter by active status."
The loop is an implementation detail. It could be a `map`, a `filter`, a list comprehension. I don't care.

## Semantic Programming
We are entering the era of Semantic Programming.
The "Source Code" is English.
The "Binary" is Python/Rust/JS.
We compile English -> Python -> Machine Code.
We are just adding one more layer to the compiler stack.

## The Loss
We lose the joy of "crafting" elegant syntax.
We gain the power to build cities instead of bricks.
It is a fair trade.

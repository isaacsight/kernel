# Agents in 3D Space

The chat interface is 2D and trapped in a window.
Spatial Computing (Vision Pro, Quest) allows us to unbox the agent.

## The Floating Cortex
Imagine coding with the Vision Pro.
*   Your code is on the main screen.
*   The **Architect Agent** is a floating node on your left.
*   The **Docs Agent** is a floating node on your right.
*   The **Guardian** is a red orb hovering above, watching for errors.

## Spatial Memory
We evolved to remember things spatially. "Where did I put my keys?"
We can map code spatially.
"The database logic is in the 'Storage Room'."
"The UI components are in the 'Gallery'."
Walking through your codebase becomes literal.

## Presence signals
In 2D, presence is a green dot.
In 3D, presence is a gaze.
The Agent avatar looks at the file it is reading.
You can see *what* the AI is paying attention to by following its gaze.
This is the ultimate interface for collaboration.

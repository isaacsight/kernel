# The Vibe Check

Engineers trust unit tests.
Humans trust "Vibes."
How do we bridge this gap?

## The Uncanny Valley of Trust
When an AI is 80% correct, it is dangerous. You trust it, and then it burns you.
When it is 99% correct, it is even more dangerous. You stop checking.

## Restoring the Gut Feeling
We built the **"Does This Feel Right?"** button.
It is an explicit signal from human to machine.
*   The code compiles. The tests pass.
*   But I click "No, it doesn't feel right."
*   The AI treats this as a generic negative reward signal and re-evaluates its approach.

## Calibration
Trust is a loop.
I learn the AI's blind spots (it sucks at CSS colors).
The AI learns my blind spots (I forget edge cases).
We cover each other. That is Symbiosis.

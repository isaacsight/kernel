# The Zone of the Thumb

The desktop is for the eyes. The mobile is for the thumb.
We redesigned our mobile agent interface to respect the **Thumb Zone**.
*   **Bottom Navigation**: All key actions (Speak, Type, Stop) are at the bottom.
*   **Card UI**: Agents present information in swipeable cards, not endless text streams.

## Influences: Perplexity & Arc
We looked at Perplexity Pro and Arc Search for inspiration.
*   **Clean Typography**: San Franciso / Inter. High contrast.
*   **Minimal Chrome**: No unnecessary headers or footers. The content is king.
*   **Fluid Physics**: Rubber-banding scrolls and snap-to-grid cards make the app feel native, not like a web view.

## The Density Problem
Coding requires high information density. Mobile screens have low density.
Solution: **Progressive Disclosure**.
*   Level 1: "Deployment Successful." (The Headline)
*   Level 2: Tap to see "5 files changed, all tests passed." (The Summary)
*   Level 3: Tap again to see the raw diffs. (The Detail)
You only see what you ask for.

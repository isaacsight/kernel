# Speed is Intelligence

If a human takes 5 seconds to answer "What is 2+2?", they look slow.
If an AI takes 5 seconds, we forgive it.
We shouldn't.

## The Flow State
Flow requires feedback loops under 200ms.
When the AI lags, it breaks Flow.
You switch tabs. You check Twitter. The moment is lost.

## Groq and the LPU
The hardware shift (LPUs, specialized inference chips) is bringing inference time down to human speeds.
This changes the *kind* of conversation you can have.
*   **Slow AI**: You give it a big task ("Write an essay").
*   **Fast AI**: You use it for micro-tasks ("Word choice," "Syntax check").

## Thinking at the Speed of Voice
When conversation is instant, you can "jam."
"Try this." "No, darker." "Too much." "Perfect."
This rapid iteration loop is only possible when latency hits zero.
Real-Time AI is not just faster; it is functionally different.

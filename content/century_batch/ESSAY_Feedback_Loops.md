# The Taste Profile

Netflix knows you like Sci-Fi. Spotify knows you like Jazz.
Your OS should know what kind of **Work** you like.

## Implicit Feedback
*   I reject 3 drafts of "Sales Copy." -> System learns I hate "Salesy" language.
*   I accept the first draft of "Technical Docs." -> System learns it is good at Docs.

## The User Model
The system builds a **User Model**.
`User prefers: Oxford Commas, Python over JS, Dark Mode, Brutalist Design.`
This model is passed to every agent.
I don't have to repeat my preferences. The system *knows* me.

## The Long-Term Alignment
Over 10 years, this model becomes a digital twin of my taste.
If I die, the system could theoretically continue to create art that looks like mine.
It captures the ineffable "Style" in a vector space.
It is the ultimate legacy.

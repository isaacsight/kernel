# Throwaway Code

We are too precious about code. We treat every line like a monument.
In **Workshop Mode**, code is clay.

## The Rapid Loop
1.  **Idea**: "A to-do list where items explode."
2.  **Prompt**: "Build this. One HTML file. Inline CSS/JS."
3.  **Result**: 30 seconds later, a working prototype.
4.  **Iterate**: "Make the explosion bigger."

## Learning by Making
I don't know if I want the feature until I play with it.
The cost of "Trying it out" used to be 4 hours.
Now it is 40 seconds.
This lowers the barrier to experimentation to near zero.

## The Junkyard
I have a folder called `prototypes/` with 500 experiments.
Most are garbage.
But the 3 that worked became the core of the Studio OS.
Quantity leads to Quality.

# The 5 Whys

Toyota invented the "5 Whys" to find the root cause of manufacturing defects.
We apply it to Idea Generation.

## The Depth Charge
User: "We need a blockchain."
Agent (Why 1): "Why do you need a blockchain?"
User: "For security."
Agent (Why 2): "Why is a standard database insecure for this use case?"
User: "I guess it isn't."
Agent: "Conclusion: We do not need a blockchain."

## Forcing First Principles
The AI is too agreeable. It assumes the user is right.
The **Socratic Agent** assumes the user is lazy.
It drills down until it hits bedrock truth.
It is annoying. But it saves millions of dollars in wasted engineering.
